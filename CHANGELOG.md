# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-11-19

### Added
- Initial release
- HURIDOCS pipeline for PDF layout analysis
- LayoutLMv3 built-in ML model support
- PyMuPDF fast extraction mode
- Transactional page-by-page processing
- LM Studio integration for local translation
- Spread detection and splitting
- Deglue operations for merged text blocks
- DOCX export with translation table
- Annotated PDF export with comments
- Comprehensive logging and metrics
- GUI mode (Tkinter)
- Docker orchestration support

### Features
- Three processing pipelines (HURIDOCS, LayoutLMv3, PyMuPDF)
- Intelligent text segmentation
- Reading order detection
- Type classification (headings, paragraphs, etc.)
- Batch translation optimization
